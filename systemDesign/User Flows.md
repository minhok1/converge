The following are all the user flows
### Group Creation
Upon creating the group, the user needs to answer these questions:
- What's the name?
- Define what this is for - there are some predefined options (sports, book club, board games, debate, learning)
- Group announcements?
- Location for each session? (Y/N)
- Time for each session? (Y/N)
- Public/Private?
- Voting system in place for each session for what/when/where? Should it be weighted? What kind of method? -> Provide a list of methods including Kemeny Young
- Is this for a competitive activity?
- If so, do you want to keep track of some sort of ranking? ELO?
- If so, do you want to track each session by points? Just win/loss? Or rankings? Is this uniform across all session? Or is each session different?
- Should users be able to rate each session and its choice?
### Session Creation
- Does this session inherit group settings?
