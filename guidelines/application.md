- This is a mobile friendly application. While this is deployed to web because of practical reasons, the styles and functionalities should be tailored for mobile screen sizes, finger tabs, draggable features, etc.
- For the entire codebase, prioritise camelCase for everything - variable names, function names, interfaces, etc
